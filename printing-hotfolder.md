# Printing - Hotfolder Default v1

Edge writes print-ready JPG files into hotfolders only AFTER order is PAID.

Hotfolders:
- /data/bpe/prints/10x15/
- /data/bpe/prints/15x20/
- /data/bpe/prints/20x30/
- /data/bpe/prints/errors/

File naming:
{order_code}_P{photoIndex}_{format}_Q{qty}_{timestamp}.jpg

Notes:
- Source images MUST be originals (no watermark).
- Previews shown to kiosks MUST be watermarked versions only.
